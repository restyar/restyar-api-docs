# 📖 مستندات فنی و راهنمای اتصال به API رستیار

به راهنمای رسمی اتصال به پلتفرم ابری و هوشمند رستیار خوش آمدید. در این مستندات، جزئیات لازم برای یکپارچه‌سازی سیستم‌های جانبی، وب‌سایت‌ها و نرم‌افزارهای حسابداری با هسته مرکزی رستیار آورده شده است.

---

## 📡 آدرس‌های پایه سرور (Base URLs)
پلتفرم رستیار دو محیط مجزا را برای تست و عملیات در اختیار شما قرار می‌دهد:

* 🧪 **محیط تست (Sandbox):** `https://sandbox-api.restyar.ir/v1` (بدون تاثیر روی دیتای واقعی رستوران)
* 🚀 **محیط عملیاتی (Production):** `https://api.restyar.ir/v1`

---

## 🧭 دامنه‌های اصلی API (Core Endpoints)

### ۱. بخش سفارش‌ها و صندوق (Cloud POS & Orders)
* `GET /orders` - دریافت لیست تمام سفارش‌های فعال و زنده کافه/رستوران.
* `POST /orders` - ثبت یک سفارش جدید از کانال‌های خارجی (مانند وب‌سایت شخصی رستوران).

### ۲. انبارداری و فرمولاسیون (Inventory & Recipes)
* `GET /inventory/items` - دریافت موجودی آنی مواد اولیه انبار.
* `PATCH /inventory/stock` - به‌روزرسانی یا کسر دستی موجودی انبار بر اساس ضایعات.

### ۳. ماژول هوش مصنوعی (AI & Forecast Insights)
* `POST /ai/forecast/sales` - دریافت پیش‌بینی هوشمند میزان فروش ۷ روز آینده بر اساس دیتای پیشین.
